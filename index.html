PDF 高质量阅读

基于 PyMuPDF 的高保真页面渲染，支持文本选择、全文搜索、流畅缩放与翻页，还原纸质文档的阅读体验。

PyMuPDF 文本选择 全文搜索

OCR 文字识别

内置 PaddleOCR，精准识别中英文混排内容。支持移动端与服务器端模型，发布版模型已预装。

AI 多模型助手

支持 OpenAI、Anthropic 及 Ollama 本地模型。选中文本即可提问、摘要、翻译或解释。

OpenAI Anthropic Ollama

Markdown 笔记

按页记录阅读笔记，支持完整 Markdown 语法和 PDF 页面引用。

会话持久化

SQLite 自动保存阅读位置、笔记和 AI 设置，下次打开无缝恢复。
